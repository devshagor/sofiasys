MEETING INFORMATION

Thursday, January 13, 2011 19:00 at SRT Solutions, 206 S. Fifth Avenue, Suite 200, Ann Arbor, MI 48104

Talks for our January meeting include:

Robin API: Simplifying asynchronous HTTP requests using block-based callbacks by Kevin Vitale of The Detroit Ketchup Company. My lightweight "Robin" toolkit allows developers to send server HTTP requests that are handled by anonymous block-based callbacks. Robin's asynchronous design pattern operates identical to Apple's own GameKit Framework.

Demonstration of a visual cooking app for the iPad and the (sometimes painful) learning process from .Net to Cocoa.

Scott Ruth will be giving an update and live demonstration of his forthcoming first app, "Time Off". It's a handy app that helps you easily keep track of how much paid time off you get from your employer, and how you've used it! This app has a lot of smart time-saving features and plenty of design polish, so come check it out!

If you are interested in talking about an iPhone or Mac OS X development topic, or showing your iPhone or Mac OS X project please contact Dave Koziol.

We're always looking for volunteers to talk at future meetings, so please consider volunteering.

INTRODUCTION

Welcome to the Ann Arbor, MI CocoaHeads Chapter! Our group includes a number of long time developers as well as developers new to the environment. We welcome all skill levels, and we discuss both Mac and iPhone topics.

Please come help build the Cocoa community in Ann Arbor!

We have a local mailing list please join it to take part in discussions and hear about upcoming meetings.
